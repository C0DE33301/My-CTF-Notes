- `imp.red`
    ```

    ;assert 1

    end
    ```
    - Written in Redcode, `;redcode`
    - The name of the program, `;name Imp Ex`
    - , `;assert 1`
    - Moves address 0 to address 1, `mov 0, 1`
    - End of program, `end`
